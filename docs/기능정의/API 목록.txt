## Weight 페이지
### Calendar 정보 조회 (Weight페이지용)
- Calendar 정보 반환
### Activity 정보 조회 (Weight페이지용)
- 날짜를 받는다
- 해당 날짜 Activity 반환
### Activity 상세페이지 조회 (Weight페이지용)
- Activity의 상세 키가 필요..
- 사진(+동영상), 시작-종료(무게, 횟수), 봉무게, 세트횟수
- 아이템 이름, 메모, 차트데이터
- 등록페이지의 경우 즐겨찾기
### Activity 저장/수정 API (Weight페이지용)
- 오늘날짜만 저장 가능 (수정의 경우 기존 데이터의 날짜를 확인해서 오늘이 아닌 경우 거부)
- 사진(+동영상), 아이템 종류, 시작-종료(무게,횟수), 봉무게, 세트횟수, 메모
